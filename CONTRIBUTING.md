# Contributing to rosette

Thank you for your interest in contributing! This is a NauroLabs research experiment — we welcome bug reports, ideas, and pull requests.

## How to contribute

1. **Report a bug** — open a GitHub Issue with steps to reproduce, expected vs actual behaviour, and your OS/browser.
2. **Propose a feature** — open an Issue first to discuss before opening a PR.
3. **Submit a PR** — fork the repo, create a feature branch, and open a pull request against main.

## Code style

- Follow the patterns already present in the codebase.
- Keep changes focused — one concern per PR.
- Update relevant documentation.

## Questions?

Open a GitHub Discussion or mention it in an Issue.
